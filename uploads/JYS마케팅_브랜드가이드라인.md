# JYS마케팅 브랜드 가이드라인

> 이 문서는 JYS마케팅의 일관된 브랜드 경험을 위한 기준을 제시합니다. 로고, 컬러, 타이포그래피, 보이스&톤, 적용 예시를 포함합니다.

---

## 1. 브랜드 아이덴티티

### 1.1 브랜드 핵심 가치
| 가치 | 설명 |
|---|---|
| 신뢰 (Trust) | 데이터와 결과로 증명하는 마케팅 |
| 친근함 (Approachable) | 어렵지 않은, 동네 사장님 눈높이의 마케팅 |
| 전문성 (Expertise) | 업종별 맞춤 전략을 설계하는 전문가 그룹 |
| 활력 (Energy) | 작은 사업의 성장을 함께 만드는 추진력 |

### 1.2 브랜드 슬로건
**"작은 기업의 큰 성장, JYS마케팅이 함께합니다"**

### 1.3 브랜드 포지셔닝
> 중소기업과 소상공인을 위한, 가장 가까운 마케팅 파트너

대형 대행사처럼 화려하지 않지만, 실질적인 성과에 집중하는 **합리적이고 든든한** 마케팅 회사로 포지셔닝합니다.

## 2. 로고 가이드라인

> ⚠️ *실제 로고 파일(AI/EPS/PNG/SVG)이 없는 상태로, 사용 규칙의 틀만 제시합니다. 로고 디자인 확정 시 본 섹션을 구체화해야 합니다.*

### 2.1 로고 구성 (제안)
- **심볼 + 워드마크** 조합형 로고 권장
- 워드마크: "JYS마케팅" (국문) / "JYS Marketing" (영문)
- 심볼: 성장/연결을 상징하는 화살표 또는 그래프 모티프 제안

### 2.2 로고 사용 규칙
- 로고 주변에는 최소 여백(로고 높이의 50% 이상)을 확보합니다.
- 로고의 가로세로 비율은 임의로 변경하지 않습니다.
- 최소 사용 크기: 디지털 기준 가로 80px 이상 (가독성 확보)

### 2.3 금지 사항 (Don't)
- ❌ 로고 색상을 임의로 변경하여 사용
- ❌ 로고에 그림자, 외곽선, 입체 효과 등 추가
- ❌ 로고를 회전하거나 왜곡하여 사용
- ❌ 배경과 대비가 약해 로고가 잘 보이지 않는 곳에 배치

### 2.4 권장 사항 (Do)
- ✅ 단색 배경에서는 주 컬러 또는 화이트/블랙 버전 사용
- ✅ 복잡한 이미지 배경에는 박스 처리 후 로고 배치
- ✅ 인쇄물/디지털 모두 벡터 원본 파일 사용

## 3. 컬러 시스템

### 3.1 주 컬러 (Primary)
| 컬러명 | HEX | 용도 |
|---|---|---|
| JYS 블루 | `#1B5FFF` | 신뢰, 메인 브랜드 컬러 (로고, 헤더, 주요 버튼) |
| JYS 다크네이비 | `#101C3A` | 텍스트, 강조 배경 |

### 3.2 보조 컬러 (Secondary)
| 컬러명 | HEX | 용도 |
|---|---|---|
| 선셋 오렌지 | `#FF7A3D` | 활력 포인트, CTA 버튼, 강조 요소 |
| 라이트 옐로우 | `#FFD166` | 보조 강조, 아이콘 포인트 |

### 3.3 뉴트럴 컬러
| 컬러명 | HEX | 용도 |
|---|---|---|
| 화이트 | `#FFFFFF` | 배경 |
| 라이트 그레이 | `#F3F5F8` | 섹션 구분 배경 |
| 미드 그레이 | `#8A93A6` | 보조 텍스트 |
| 다크 그레이 | `#2B2F38` | 본문 텍스트 |

### 3.4 컬러 사용 비율 가이드
- 주 컬러(블루) 60% : 뉴트럴 30% : 포인트 컬러(오렌지/옐로우) 10%
- 포인트 컬러는 CTA, 강조 텍스트 등 **시선을 유도해야 하는 곳**에만 제한적으로 사용

## 4. 타이포그래피

### 4.1 국문 폰트
- **제목/헤드라인**: Pretendard Bold / SemiBold
- **본문**: Pretendard Regular / Medium
- (대안: Noto Sans KR)

### 4.2 영문 폰트
- **제목**: Montserrat SemiBold
- **본문**: Inter Regular

### 4.3 타이포 스케일 (웹 기준 예시)
| 용도 | 크기 | 굵기 |
|---|---|---|
| H1 (메인 타이틀) | 32~40px | Bold |
| H2 (섹션 제목) | 24~28px | SemiBold |
| H3 (소제목) | 18~20px | Medium |
| Body (본문) | 15~16px | Regular |
| Caption (보조설명) | 12~13px | Regular |

### 4.4 타이포 사용 원칙
- 한 화면(또는 한 문서)에 폰트는 국문/영문 각 1종 이내로 제한
- 강조는 굵기(Bold)로, 색상 강조는 포인트 컬러로만 제한적으로 사용
- 본문 줄간격(line-height)은 150~160% 권장 (가독성 확보)

## 5. 보이스 & 톤 (Voice & Tone)

### 5.1 브랜드 보이스 원칙
| 원칙 | 설명 |
|---|---|
| 쉽게 말하기 | 마케팅 전문 용어를 풀어서 설명 |
| 진심으로 말하기 | 과장된 약속보다 현실적인 기대치 전달 |
| 함께 말하기 | "우리가 해드립니다"보다 "함께 만들어갑니다" 톤 유지 |
| 자신감 있게 말하기 | 전문성을 바탕으로 명확하고 단정한 어조 사용 |

### 5.2 톤 예시

**✅ 권장 표현**
- "사장님의 매장에 꼭 맞는 마케팅 전략을 함께 찾아드릴게요."
- "이번 달 광고 성과, 숫자로 정리해서 보여드립니다."

**❌ 지양 표현**
- "무조건 매출 10배 보장!" (과장된 약속)
- "본 업체는 최첨단 AI 알고리즘을 통해…" (불필요하게 어려운 표현)

### 5.3 채널별 톤 가이드
| 채널 | 톤 | 비고 |
|---|---|---|
| 홈페이지/제안서 | 신뢰감 있고 전문적인 톤 | 데이터·근거 중심 |
| SNS(인스타그램 등) | 친근하고 캐주얼한 톤 | 이모지/구어체 일부 허용 |
| 고객 응대(상담, 카카오톡) | 따뜻하고 친절한 톤 | 존댓말 기본, 과도한 격식 지양 |
| 리포트/제안 문서 | 명확하고 간결한 톤 | 불필요한 수식어 최소화 |

## 6. 이미지 & 비주얼 스타일

### 6.1 사진 스타일
- 실제 매장, 실제 인물 중심의 **현장감 있는 사진** 우선
- 과도하게 보정되거나 인위적인 스톡 이미지 지양
- 밝고 자연광 느낌의 톤(따뜻한 화이트 밸런스) 권장

### 6.2 아이콘/그래픽 스타일
- 라인(Outline) 또는 단색 플랫 아이콘 사용
- 둥근 모서리(Rounded) 형태로 친근한 느낌 유지
- 아이콘 컬러는 주 컬러 또는 포인트 컬러 1색 사용 원칙

### 6.3 데이터 시각화(차트 등)
- 그래프 메인 컬러는 JYS 블루, 강조 항목만 오렌지 사용
- 불필요한 3D 효과, 그림자 지양 → 단순하고 명확하게

| 적용 대상 | 가이드 |
|---|---|

## 9. 추가로 필요한 정보 (확정 시 보완 필요)

- 실제 로고 파일 및 심볼 디자인 확정안
- 정확한 브랜드 컬러(이미 정해진 컬러가 있다면 HEX 코드)
- 보유 폰트 라이선스 여부 (Pretendard 등 무료 폰트 외 별도 폰트 사용 시)
- 인쇄물(명함, 브로슈어 등) 별도 규격 필요 여부
- 영문 브랜드명 표기 방식 확정 (JYS Marketing / JYS MARKETING 등)
